## breadcrumb 面包屑

### 基础用法

:::demo 
```html
  <lv-breadcrumb></lv-breadcrumb>
```
:::

##### 特别注意,需要与vue-router配合使用

```js
var router = [
  {
    path: '/doc/breadcrumb',
    name: 'breadcrumb',
    meta: {
      title: 'breadcrumb 面包屑'
    }
  }
]
```
